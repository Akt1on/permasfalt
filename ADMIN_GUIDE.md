# Руководство администратора — Пермь Асфальт 59

## Первый вход

1. Открыть `https://permasfalt59.ru/auth`
2. Нажать «Создать аккаунт»
3. Email: **permsite1@gmail.com** (прописан в базе как администратор)
4. Подтвердить email → войти → открывается `/admin`

> Изменить email администратора можно в Supabase SQL Editor (см. `supabase/README.md`)

## Разделы административной панели

### 📊 Обзор (дашборд)
- Счётчики: новые заявки, всего заявок, услуги, проекты
- Последние 6 заявок с быстрым доступом
- Живое обновление каждые 30 секунд

### 📥 Заявки (CRM)
- Поиск по имени, телефону, услуге
- Фильтр по статусам: Новая / В работе / Готово / Архив
- Изменение статуса — кликом по кнопке-пилюле
- Копирование телефона одним нажатием
- Время заявки: дата + "N часов назад"

### 🔧 Услуги
- CRUD полный: создать, изменить, удалить
- SEO-поля: hero-подпись, полное описание
- **Что входит**: каждый пункт с новой строки
- **FAQ**: формат `Вопрос | Ответ` (каждая пара с новой строки)
- Детальный прайс по услуге (отдельные позиции)
- Активность: вкл/выкл без удаления

### 🗂 Портфолио
- CRUD проектов: название, категория, локация, описание
- Фотогалерея к каждому проекту (drag-drop загрузка)
- Обложка проекта
- Скрыть/показать без удаления

### 🖼 Галерея
- Фотографии по 8 категориям: Дороги, Парковки, СНТ/дачи, Склады, Тротуарная плитка, Промышленные, Дворы, Прочее
- Фильтр с счётчиком фото в каждой категории
- Порядковый номер (для управления порядком)

### 💰 Прайс-лист
- Inline-редактирование: клик → редактировать → Enter
- Добавить / удалить строку в категории
- Добавить / удалить категорию
- Одна кнопка «Сохранить всё» (пакетное обновление)
- Индикатор несохранённых изменений

### ⭐ Отзывы
- CRUD с рейтингом (1–5 звёзд, интерактивный выбор)
- Поля: имя, должность, компания, источник, аватар
- Показать/скрыть без удаления

### 📝 Блог
- Заголовок, slug, excerpt, контент (Markdown/HTML)
- Ключевые слова, время чтения, обложка
- Черновик / Опубликовать

### ⚙️ Настройки
Пять секций, каждая сохраняется отдельно:
1. **Контакты** — телефоны, email, адрес, часы работы
2. **Мессенджеры** — WhatsApp, Telegram, ВКонтакте, MAX (с проверкой ссылки)
3. **Главный экран** — бейдж, заголовок, подзаголовок
4. **О компании** — заголовок и текст раздела
5. **Реквизиты** — юридическое название, ОГРН, ИНН

## Загрузка фотографий

- Форматы: JPG, PNG, WebP, GIF
- Размер: до 10 МБ
- Поддержка drag-and-drop
- Хранятся в Supabase Storage bucket **`site-images`**
- После загрузки URL копируется автоматически
